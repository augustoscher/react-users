# react-users

## Setup

### Backend
1. Run backend:
``` 
cd backend
npm start

``` 
### Frontend
1. Run frontend:
``` 
cd frontend
npm start

``` 

## Testing

### backend routes
``` 
GET http://localhost:3002/users
GET http://localhost:3002/users/1
POST http://localhost:3002/users 
PUT http://localhost:3002/users
```
